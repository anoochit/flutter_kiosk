# Flutter Kiosk

A sample kiosk app inspired by [Cuberto's design](https://www.behance.net/gallery/84835219/McDonalds-Kiosk-Redesign). The kiosk has 2 parts

 1. Strapi for backend and Restful API
 2. Kiosk app
